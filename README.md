# ZaAndPoint (Зачётная Книжка и точка)

## О проекте

**ZaAndPoint** – программное обеспечение для создания, обработки и просмотра данных по аттестации студентов по дисциплинам семестра.

Программа (клиентская часть) реализована на языке Python. Работает в любой среде, совместимой с Windows. Дисковой памяти для запуска программы требуется не менее 200 Mb. Оперативной памяти для нормальной работы программы требуется не менее 1 Gb. Имеет графический интерфейс. Программа связана с серверной частью, написанной на PHP, которая работает с БД MySQL.

## Назначение

Программа «Зачётная Книжка и точка» («ZaAndPoint») предназначена для организации и обработки данных о результатах аттестации, проводимой по дисциплинам семестра. Она обеспечивает удобный доступ к информации о результатах аттестации для преподавателей и студентов, позволяя им просматривать отчеты и анализировать данные. Администрация получает доступ к управлению всей базой данных, что позволяет эффективно контролировать и управлять процессом аттестации.

## Пользователи

Программа используется студентами, преподавателями и администрацией.

## Возможности

### Студенты
- Просмотр своих оценок за все семестры
- Фильтрация оценок по семестрам
- Просмотр контактов преподавателей
- Просмотр информации о своей группе и одногруппниках

### Преподаватели
- Выставление и редактирование оценок
- Просмотр успеваемости студентов своих групп
- Просмотр контактов сотрудников

### Администрация
- Управление пользователями (добавление, редактирование, удаление)
- Управление направлениями подготовки
- Управление группами студентов
- Управление дисциплинами
- Управление связями (преподаватель-дисциплина, дисциплина-направление и др.)
- Подтверждение регистрации новых пользователей

### Общие функции
- Просмотр таблиц с возможностью фильтрации
- Добавление новых записей
- Редактирование существующих записей
- Удаление записей
- Экспорт данных в Excel
- Автоматическое обновление данных

## Установка

### Клиентская часть

1. Скопировать папку `client` из репозитория
2. В файле `config.py` изменить `BASE_URL` на рабочий URL сервера
3. Установить Python 3.8 или выше
4. Установить зависимости:
   ```bash
   pip install PySide6 requests pandas
   ```
5. Запустить программу:
   ```bash
   python main.py
   ```

### Серверная часть

1. Скопировать папку `server` в корневую директорию веб-сервера
2. Установить PHPMyAdmin
3. Создать базу данных `gradestudent`
4. Импортировать структуру БД из файла `sbbd.sql`
5. Создать пользователей БД:
   - `view_user` – для авторизации
   - `student_user` – для студентов
   - `teacher_user` – для преподавателей
   - `secretary_user` – для секретарей
6. В PHP-файлах изменить пароли подключения к БД на действующие
7. Реализовать Apache

## Структура базы данных

| Таблица | Назначение |
|---------|------------|
| User | Пользователи системы |
| TempUser | Временные пользователи (ожидают подтверждения) |
| Student | Студенты |
| Teacher | Преподаватели |
| Secretary | Секретари |
| Direction | Направления подготовки |
| StudentGroup | Учебные группы |
| StudentGroupLink | Связь студентов с группами |
| Discipline | Дисциплины |
| DisciplineDirection | Связь дисциплин с направлениями |
| TeacherDiscipline | Связь преподавателей с дисциплинами |
| DirectionTeacher | Связь направлений с преподавателями |
| Grades | Оценки (аттестация) |

## Безопасность

- Пароли хранятся в хэшированном виде (bcrypt)
- На стороне клиента – SHA-256
- Четыре уровня доступа к БД
- Защита от SQL-инъекций
- Защита от XSS
- Лимит 10 запросов/час с одного IP
